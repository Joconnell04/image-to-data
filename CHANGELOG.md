# Image to Data Changelog

## [1.0.0] - 2026-02-02

### Added

- Initial release
- Interactive screen capture with macOS screenshot tool
- Automatic content type detection (table, chart, diagram, general)
- OpenAI vision model integration for data extraction
- TSV output for tables
- Structured output for charts with axis labels and data points
- Component and relationship extraction for diagrams
- OCR-like text extraction for general images
- Configurable AI models for routing and extraction
- Optional confidence score in output
- Force mode option to override automatic detection
